# UBIG 세미 프로젝트 초기 데이터 및 테스트 환경 (Environment Setup)

> **프로젝트 시연 및 개발을 위한 초기 더미 데이터와 테스트 계정 정보입니다.**

---

## ✅ 1. 초기 더미 데이터 삽입 현황

시스템 초기 구동 및 기능 테스트를 위해 주입된 기초 데이터 통계입니다.

| 테이블 | 건수 | 주요 내용 |
|---|---|---|
| **MEMBERS** | 7명 | 관리자(admin) 및 테스트 유저(user01~06) |
| **ANIMAL_DETAILS** | 20마리 | 강아지 10마리 + 고양이 10마리 프로필 |
| **ADOPTION_POSTS** | 20건 | 각 동물별 1:1 홍보 게시글 연동 |
| **ACTIVITIES** | 10건 | 완료/모집중/예정 상태의 봉사 프로그램 |
| **SIGNS** | 15건 | 회원들의 봉사 신청 및 승인 내역 |
| **FUNDINGS** | 8건 | 진행 중인 크라우드 펀딩 프로젝트 |
| **BOARDS** | 20건 | 공지사항, 자유게시판, 후기 데이터 |
| **COMMENTS** | 20건 | 게시글별 댓글 및 대댓글 데이터 |

---

## 🔑 2. 테스트 계정 정보 (Test Accounts)

시연 및 기능 심사를 위해 준비된 테스트 계정 목록입니다.

| USER_ID | 비밀번호(원문) | ROLE | 비고 |
|---|---|---|---|
| **admin** | `admin` | ADMIN | 시스템 관리자 |
| **user01** | `pass01` | USER | 일반 사용자 01 |
| **user02** | `pass01` | USER | 일반 사용자 02 |
| **user03** | `pass01` | USER | 김민지 (입양 신청 테스트용) |
| **user04** | `pass01` | USER | 박준혁 (봉사 활동 테스트용) |
| **user05** | `pass01` | USER | 이수연 (펀딩 후원 테스트용) |
| **user06** | `pass01` | USER | 최지호 (커뮤니티 테스트용) |

> **Security Note**  
> 모든 비밀번호는 `BCrypt` 10 rounds 암호화되어 저장되어 있으며, 위 정보는 시연을 위한 원문 정보입니다.

## 🛠️ 3. 데이터 초기화 가이드
- **Docker 환경**: `init_db.sql` 파일을 통해 컨테이너 생성 시 자동으로 테이블 생성 및 데이터가 주입됩니다.
- **수동 초기화**: DB 도구(DBeaver 등)에서 프로젝트 루트의 `init_db.sql` 스크립트를 실행하여 초기 상태로 복구할 수 있습니다.
